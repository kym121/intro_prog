Tree like structure

HTML document can be expressed. The html tag is tree's root, head tag and body tag are stems. h1 tag, img tag and other in body tag is tree's leaf. This tree like structure called "DOM"(Document Object Model).

HTML-CSS-DOM

HTML, CSS, DOM is made a web page.

HTML is language that is expressed a structure of a webpage. HTML tag is elements in tree.

CSS(Cascading Style Sheets) is language that is expressed a style(font color, font size, background color, and other) of a webpage.

DOM is a structure of a webpage that is expressed in HTML.

Boxes-Everywhere

All shape(title, circle, and other) is managed as squares in HTML document. There is smaller box in the biger box.
