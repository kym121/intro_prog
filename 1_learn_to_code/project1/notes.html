Thinking like a Programmer

The Purpose of this Nanodegree is to practice thinking like a programmer. It's a following five ideas.

Procedual Thinking

A computer can't understand things that a human can do easy. So, We tell a computer procedure to detail. Once telled it, A computer can do faster than a human.

Abstract Thinking

Abstract Thinking is to treat similarity things by same method. Therefore, we can write shorter code than not to use abstract thinking.

Systems Thinking

Systems Thinking is to devide a big problem into smaller multiple problem. It is useful to create a plan to solve a big problem.

Technological Empathy

Technological Empathy is basic understanding. For example, computer empathy is to understand a computer's ability and princeple.

Systematic Debugging

Debugging is a process that we do when computer program doesn't work. When a computer program doesn't work, there is a mistake in a code. We must find a mistake, and repair it.

World Wide Web

World Wide Web(WWW) is collections of HTML document. It's witten by "HyperText Markup Language(HTML)" and included links to videos, images, music, other document and other.

WWW is cofigured in a web browser, the Internet and servers(includes of HTML documents). A web browser request via the Internet to servers, and servers response files that web browser displays.

HTML basics

HTML can be written following ideas.

  • Text content
  • Markup(What it looks like)
  • Refer to videos, images, other document and other
  • Links to other page

HTML markup

The HTML is described as follows. HTML used tags(<,>).

<OPENING TAG > CONTENTS </CLOSING TAG >

For example, If you want to indicate text in bold, you written as following.

< b > text < /b >

Samples of HTML Markups

<b>TEXT</b> "TEXT" is displayed in Bold.
<em>TEXT</em> "TEXT" is displayed in Italic.
<a href="URL">TEXT</a> "TEXT" is displayed, and if "TEXT" is clicked, the browser is displayed the page of "URL".
<img src="PATH"> The image of "PATH" is displayed.
<br> Line break
<p>TEXT</p> "TEXT" is a one paragraph.
<span>CONTENTS</span> "CONTENTS" is inline, and style is setted by CSS.
<div>CONTENTS</div> "CONTENTS" is block, and style is setted by CSS.
