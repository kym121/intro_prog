World Wide Web

World Wide Web(WWW) is collections of HTML document. It's witten by "HyperText Markup Language(HTML)" and included links to videos, images, music, other document and other.

WWW is cofigured in a web browser, the Internet and servers(includes of HTML documents). A web browser request via the Internet to servers, and servers response files that web browser displays.

HTML basics

HTML can be written following ideas.

  • Text content
  • Markup(What it looks like)
  • Refer to videos, images, other document and other
  • Links to other page

HTML markup

The HTML is described as follows.

<OPENING TAG > CONTENTS </CLOSING TAG >

For example, If you want to indicate text in bold, you written as following.

< b > text < /b >

Samples of HTML Markups

  • b: Text in bold
  • em: Text in Italic
  • a: Link to other page
  • img: Image
  • br: Line break
  • p: Paragraph
  • span: Use inline for CSS
  • div: Use block for CSS
